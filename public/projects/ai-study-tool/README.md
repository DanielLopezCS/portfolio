# AI Study Tool Project Images

Place your 3 project images in this folder with the following names:

- `image1.jpg` (or `.png`) - First image showcasing the syllabus upload feature
- `image2.jpg` (or `.png`) - Second image showcasing flash cards
- `image3.jpg` (or `.png`) - Third image showcasing practice exams/problems

The carousel will automatically load these images. If images are not found, placeholder images will be shown.

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
