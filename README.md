# Portfolio

Modern portfolio website built with React, TypeScript, and Vite.

## Features

- Modern, sleek design with smooth animations
- Responsive layout for all devices
- Sections: Hero, Work Experience, Education, Skills, Projects, and Contact
- Interactive carousel for project showcase
- Smooth scrolling navigation

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion (animations)
- CSS with modern design patterns

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Project Structure

```
src/
  components/
    - Hero.tsx
    - Navigation.tsx
    - WorkExperience.tsx
    - Education.tsx
    - Skills.tsx
    - Projects.tsx
    - Contact.tsx
```
